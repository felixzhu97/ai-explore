# MCP UI/交互对比检查清单

## 使用方式
通过 cursor-ide-browser MCP 工具手动/半自动执行以下检查项，完成后勾选 `- [x]`

## 检查项

### AI Hub - Chat 标签页
- [ ] 页面加载无控制台错误
- [ ] Chat 标签默认激活
- [ ] 空状态显示正确（图标、标题、描述）
- [ ] Quick Action 按钮可见且可点击
- [ ] Quick Action 点击后填入输入框
- [ ] 发送按钮在空输入时禁用（opacity: 0.5）
- [ ] 输入文字后发送按钮启用
- [ ] 发送消息后用户消息显示正确样式
- [ ] Tab 切换（Chat → Image → TTS）正常

### AI Hub - Image 标签页
- [ ] Prompt 输入区域正常
- [ ] 尺寸选择器（512x512, 768x768, 1024x1024）可切换
- [ ] 生成按钮在空 Prompt 时禁用
- [ ] 生成中显示 loading overlay
- [ ] 生成结果图片可点击放大
- [ ] 下载按钮正常

### AI Hub - TTS 标签页
- [ ] 语音选择下拉框正常
- [ ] 语速滑块可调节
- [ ] 合成按钮在空文本时禁用
- [ ] 合成中显示 spinner
- [ ] 音频播放器显示且可播放/暂停
- [ ] 下载音频按钮正常

### Vision Panel
- [ ] 页面加载无错误
- [ ] 图片上传区域正常
- [ ] 分析按钮正常

### 全局 UI 一致性
- [ ] 所有面板圆角一致（12px）
- [ ] 主色调一致（#0071e3）
- [ ] 字体大小和间距符合设计规范
- [ ] 响应式布局正常（移动端/平板/桌面）
- [ ] 深色/浅色模式切换正常（如支持）
